## ğŸš€ Getting Started

### Prerequisites
- Node.js >= 14.x
- npm >= 6.x
- Expo CLI (`npm install -g expo-cli`)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/laurentiucretu68/battleships
   cd battleships
   ```
   
2. Install the dependencies:
   ```bash
   npm install
    ```

3. Start the project:
    ```bash
    npm start
    ```
